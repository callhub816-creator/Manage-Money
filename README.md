# Money Machine - Indian Finance Guide

Your comprehensive platform for comparing financial products in India including credit cards, loans, and insurance.

## Features

- **Credit Cards**: Compare the best credit cards from major Indian banks
- **Loans**: Find quick disbursing loans, low CIBIL score loans, and business loans
- **Insurance**: Explore health, term, car, and bike insurance options
- **CIBIL Score**: Understand your credit score
- **Calculators**: EMI, SIP, and FD calculators

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
